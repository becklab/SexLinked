Most data was taken from The Cancer Genome Atlas (TCGA) via either the Genome Data Analysis Center at the Broad Institute or cBioPortal.

TCGA RNASeqv2 Expression Data

This file has all of the RNA expression data used for differential gene expression analysis. The relevant script is here. The file is in .RData format. Note that this data belongs to TCGA, but we have merely coerced it into a format compatible with our analyses.

Download the file.

TCGA Clinical Data

This file has all of the clinical annotations for patients in TCGA. The file is referenced by numerous scripts in copy number and methylation analyses. The file is in .RDS format. Note that this data belongs to TCGA, but we have merely coerced it into a format compatible with our analyses.

Download the file.

Human Gene List

This file contains aliases for HUGO identifiers. It can be used in conjunction with the methylation and copy number analyses to narrow the list of genes being investigated.

Download the file.

Lincscloud Input Files

These files have all the inputs to Lincscloud. There are 8 files per cancer. 4 contain Affymetrix IDs, which Lincscloud does not accept but other connectivity map services may. The other 4 contain converted HUGO IDs. There are 2 files per sex, one indicating genes up (i.e., resistance signature) and the other indicating genes down (i.e., sensitivity signature) in tumors. Note that when inputting lists to Lincscloud, the resistance signature ought to be listed under genes down, since you are looking for a perturbagen that down-regulates the resistance signature. All 8 files are contained in folders separated by cancer type.

Head and Neck Squamous Cell Carcinoma (HNSC)
Kidney Chromophobe (KICH)
Kidney Renal Clear Cell Carcinoma (KIRC)
Liver Hepatocellular Carcinoma (LIHC)
Lung Adenocarcinoma (LUAD)
Lung Squamous Cell Carcinoma (LUSC)
Thyroid Carcinoma (THCA)
